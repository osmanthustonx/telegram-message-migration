# Product Overview

Telegram 訊息遷移工具：將帳號 A 的對話訊息安全遷移至帳號 B 可存取的群組。

## Core Capabilities

1. **對話列舉與過濾**：自動列舉帳號 A 的所有對話，支援按 ID、類型、日期範圍過濾
2. **群組自動建立**：為每個來源對話建立對應的目標群組，並邀請帳號 B 加入
3. **批次訊息轉發**：以批次方式轉發訊息，保留原始時序
4. **斷點續傳**：持久化遷移進度，支援中斷後繼續遷移
5. **流量控制**：智慧處理 Telegram FloodWait 限制，自適應調整請求速率

## Target Use Cases

- **帳號遷移**：將舊帳號的重要對話內容遷移至新帳號
- **資料備份**：透過群組方式保存對話歷史，讓多個帳號可存取
- **組織遷移**：企業或團隊帳號變更時的批次訊息轉移

## Value Proposition

- **自動化**：無需手動逐一轉發訊息，一次指令完成全部遷移
- **可靠性**：斷點續傳確保大量訊息遷移不會因中斷而遺失進度
- **合規性**：使用 Telegram 官方 MTProto API (GramJS)，符合平台規範
- **可追蹤**：完整的進度報告與日誌記錄，清楚掌握遷移狀態
